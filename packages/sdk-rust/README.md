# autonex-sdk-rust

Solana-compatible Rust crate for AUTONEX SDK domain types, deterministic serialization, and validation.

Focus:
- Strong typing
- Explicit error handling
- Deterministic serialization (Borsh)
- No async runtime assumptions

## Modules
- agent.rs
- intent.rs
- policy.rs
- execution.rs
- errors.rs
- validation.rs

## Dev

```bash
cargo check
cargo test
```
